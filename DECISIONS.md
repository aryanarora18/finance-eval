## Phase notes

### Phase 1 — data + golden set (done)
- EDGAR R-file renderings (via FilingSummary.xml) give per-statement tables as clean text — no scraping, no PDF parsing. Provenance in `data/raw/manifest.json`.
- Every golden primary + alt recomputed programmatically from raw line items: 38/38 checks pass.
- The golden set is where the finance judgment lives: which lines count as debt (commercial paper: yes; operating leases: only disclosed), what EBITDA does NOT add back (impairments, SBC, rent), when refusal is the right answer (DAL gross margin), what must be disclosed to be trustworthy (NVDA's $8.9B paper gains = 8% of EBITDA).

### Phase 2 — harness (done, offline-validated)
- Tier-1 checks: schema, value-vs-primary/alts, unit+scale (percent-as-decimal, billions/millions), verbatim-normalized grounding of cited inputs, determinability match, prior-period hit.
- 18/18 offline meta-eval checks pass (planted defects all caught; correct answers all pass).

### Phase 3 — live run + findings (done, 2026-07-03)
- Full sweep executed: 2 models × 5 companies × 6 metrics × 3 runs = 174 gradeable metric-runs (+30 judge calls). Checkpoint/resume held; one cell lost to a live JSON parse failure (Sonnet/COST run1), absorbed cleanly.
- Aggregate: Tier-1 value accuracy 97% (169/174), finance-aware pass 93% (162/174). "Right number, failed judgment" = 7 metric-runs.
- Full writeup in `findings/FINDINGS.md`. Headlines: AAPL net debt 0/6 (method incoherence + $78B method swing, undisclosed); Tier-2 judge is non-deterministic on identical answers; 3 of 5 designed probes came back clean (fabrication, memorization, KR impairment).
